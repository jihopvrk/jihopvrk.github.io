brainstorm

Brainstorm is an ongoing project where I look into the specifics of the fashion industry. In this day and age it feels like fashion is always being repeated and everything looks the same. Using elements of recognizable brands, I create shoes to show that the industry is somewhat boring.

glitch

dead or alive

2019

web

Although the title can seem extreme for what the website is, this piece is called dead or alive because overthinking has always been a serious problem for me. The twisted thing about struggling through this is that every time I think that I'm overthinking I'll tend to overthink more just based on overthinking. That is a very confusing sentence to say the least, but it sums up my feelings about overthinking.

cookie

2019

web

The concept of this website is that you have to be careful on the internet. Although this is a very basic concept, this website uses both PSA elements as well as puzzle elements to make the website interactive in a fun way (hopefully). Browsing the internet safely is very important in order to make sure your personal information does not get leaked among other bad things.
